**introducing anchor-it-here**: init anchor workspace from anywhere.


This CLI tool allows you to: Initialize a new Anchor workspace in your current working directory, Avoid `anchor init project_name` and folder reorganization, Save time and stay in flow

you can view the [tweet here](https://x.com/avhidotsol/status/1921878222720901597). the code is open-sourced and available on [github](https://github.com/AvhiMaz/anchor-it-here), check the package [here](https://www.npmjs.com/package/anchor-it-here)
